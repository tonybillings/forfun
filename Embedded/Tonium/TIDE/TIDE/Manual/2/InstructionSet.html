Instruction Set
Arguments named DEST (destination), SRC (source), and OP1/OP2 (operator 1/2) are register addresses while CONST (constant) arguments are numeral literals.

OperationArg1Arg2Arg3Description
NOP------No operation. Used to delay the processor core by one clock cycle.
INDESTOP1CONSTRead value from OP1 memory-mapped address (plus CONST offset) and store in DEST register.
INCDESTCONST--Read value from CONST memory-mapped address and store in DEST register.
OUTSRCOP1CONSTWrite value in SRC register to OP1 memory-mapped address (plus CONST offset).
OUTCSRCCONST--Write value in SRC register to CONST memory-mapped address.
SETDESTOP1--Write value in OP1 register to DEST register.
SETCDESTCONST--Write CONST value to DEST register.
