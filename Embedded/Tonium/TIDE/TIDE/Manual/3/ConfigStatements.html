xT50 :: Config Statements
Config statements are used to set configuration parameters, which are described and listed here. These statements begin and end with an opening and closing square bracket, respectively. They follow typical assignment-style syntax, with the config parameter on the left of an assignment operator and its assigned value on right of the operator, as illustrated below:

[CLOCK = 120]
[P31 = INPUT_PULLDOWN, 248]
[P32 = OUTPUT, 249]

Note that no aspect of a config statement is case-sensitive and the spaces are optional:

[clock=120]
[p31=input,248]
