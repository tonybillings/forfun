xT50 :: Numeral Literals
Numeral literals can be specified in binary, decimal, and hexa-decimal format. As to be expected when working with an 8-bit processor, the valid range for a value is between 0 and 255. The following three instructions are equivalent:

SET R1 01111010b
SET R1 122
SET R1 0x7A

In the example above, the three instructions specifiy the same value in binary, decimal, and hexa-decimal format, respectively. Note that the 'b' and 'x' characters in the binary and hexa-decimal numeral literals are not case-sensitive.
